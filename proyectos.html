Urban Routes – App de Car-Sharing

Descripción simple:
Urban Routes permite a los usuarios reservar taxis fácilmente. Mi tarea fue probar automáticamente que todo el flujo de reserva funcionara correctamente: elegir ruta, tipo de servicio, registrar teléfono, pago y extras como mantas o helados.

Detalles técnicos:
Automatización de pruebas de extremo a extremo (E2E) con Python y Selenium WebDriver, usando Page Object Model (POM) para mantener código limpio y escalable. Cobertura completa de escenarios críticos y registro de resultados mediante logs.
