Proyecto Urban Routes – App de Car-Sharing

Urban Routes es una aplicación web que redefine la manera de solicitar servicios de transporte. Con **planificación inteligente de rutas**, selección de tarifas, comunicación directa con el conductor y la posibilidad de agregar servicios extra como mantas, pañuelos o helados, Urban Routes ofrece una experiencia de movilidad completa y personalizada. El proyecto de **automatización de pruebas end-to-end (E2E)** permite validar de forma confiable cada paso del flujo de reserva de un taxi, desde la configuración de la ruta hasta la confirmación del viaje, asegurando que la aplicación funcione sin errores en escenarios reales de uso. Gracias a **Python**, **Selenium WebDriver** y el patrón **Page Object Model (POM)**, la suite de pruebas es escalable, mantenible y robusta, cubriendo la interacción con la interfaz, la lógica de negocio y servicios adicionales. 👉 Explora el repositorio para ver cómo se implementa la automatización completa, desde la selección de direcciones y tipo de transporte hasta la validación de pagos y comunicación con el conductor.

Proyecto Urban Grocers – API Automation

Urban Grocers es un proyecto de automatización de pruebas para una API de servicios de entrega, diseñado para validar la lógica compleja de un sistema real de e-commerce. El proyecto pone a prueba funcionalidades clave como la creación de pedidos, gestión de carritos, generación de kits personalizados y la integración con múltiples servicios de entrega y almacenes. A través de pruebas automatizadas en **Python**, se verifican reglas de negocio críticas como validaciones de datos, selección óptima de proveedores y cálculo dinámico de costos. Este proyecto destaca por su enfoque en **testing de caja negra**, cubriendo escenarios reales, casos límite y validaciones robustas que garantizan la confiabilidad del sistema. 👉 Explora el repositorio para ver la implementación completa, estructura modular y casos de prueba que simulan el comportamiento de una plataforma de delivery a escala.

Proyecto Urban Scooter

# Urban Scooter Urban Scooter es una aplicación móvil que facilita el alquiler de scooters eléctricos por varios días, gestionando pedidos, entregas y notificaciones de manera eficiente. Con control de pedidos por ubicación, filtros por estación de metro y gestión de conectividad offline, Urban Scooter optimiza la experiencia tanto para usuarios como para repartidores. El proyecto de **testing profesional** permite validar de forma confiable todas las funcionalidades críticas de la app: desde el inicio de sesión y aceptación de pedidos, hasta notificaciones, cancelaciones y finalización de entregas, asegurando que la aplicación funcione correctamente en escenarios reales de uso. Gracias a herramientas como **Postman**, **emuladores móviles**, pruebas **UI manuales**, y **SQL** para la gestión de datos, junto con técnicas de diseño de pruebas como **clases de equivalencia**, **valores límite** y **tablas de decisión**, el proyecto garantiza cobertura completa y documentación profesional de cada escenario. 👉 Explora el repositorio para ver cómo se implementan las pruebas sobre la app móvil, desde la validación de campos y notificaciones, hasta la gestión de pedidos, conectividad offline y filtros por estación de metro.

Mercado Libre – Botón “Agregar al carrito”

# QA Mobile Testing Portfolio QA Mobile Testing Portfolio es un proyecto de **testing profesional** enfocado en aplicaciones móviles, diseñado para validar de forma confiable la experiencia de usuario y las funcionalidades críticas en dispositivos como **iPhone 13** y **Samsung Galaxy A17 5G**. El proyecto documenta casos de prueba y bugs relacionados con **acciones accidentales**, zonas táctiles críticas, interacción con botones durante scroll y recuperación de errores, asegurando que los usuarios puedan navegar y comprar sin fricciones ni frustraciones. Cada bug y test case incluye pasos claros, resultados actuales y esperados, evidencia en video y capturas de pantalla, y análisis detallado de impacto y frecuencia, para que cualquier equipo de desarrollo pueda reproducir y corregir los problemas con eficiencia. Gracias a herramientas como **MantisBT**, **GitHub**, pruebas manuales en múltiples dispositivos y técnicas de diseño de pruebas como **pruebas exploratorias**, **cross-device testing** y **verificación de UX**, el proyecto garantiza cobertura completa y documentación profesional de cada escenario crítico. 👉 Explora el [repositorio del proyecto](https://github.com/TU-USUARIO/qa-mobile-testing-portfolio) para ver cómo se implementan los tests sobre la app móvil, desde la validación de interacción accidental y tamaño de hit area, hasta la gestión de errores y flujos de recuperación, asegurando una experiencia de usuario optimizada en escenarios reales de uso.
